# Simple Resume Builder

A dynamic, front-end web application that allows users to seamlessly create, edit, and export professional resumes directly from their browser.

## Repository Details

**Suggested Description:**
An engaging, feature-rich interactive resume builder featuring drag-and-drop mechanics, inline robust editing, customizable styling, browser local storage saving, and high-fidelity PDF export capabilities.

**Suggested Topics/Tags:**
resume-builder, drag-and-drop, interactive-ui, pdf-generation, vanilla-javascript, frontend, web-app, github-pages

## Deployment

The application is deployed and hosted on GitHub Pages:
[Live Application](https://aaqibhafeezkhan.github.io/SimpleResumeBuilder/)

## Features

- Drag and Drop Interface: Intuitively build resume structures by dragging components onto the main document canvas.
- Inline Editing: Seamlessly edit document content with robust contenteditable fields.
- Premium Theming: Toggle between curated color palettes and professional typography options to match visual preferences.
- Local Storage Support: Continuous auto-save ensures uninterrupted work, persisting the document state locally.
- Advanced PDF Export: High-quality, precise printing logic ensures the layout retains structure and vector selection capability when exporting to PDF.
- Responsive Design: Modular sidebar and dynamic center stage optimized for workflow efficiency.

## Setup and Development

1. Clone the repository.
2. Open the index.html file in any modern web browser to utilize the application.
3. No build tools or node dependencies are required for development.

## Technologies Used

- Vanilla HTML5
- Modern CSS3 (Variables, Grid, Flexbox, Keyframes)
- Vanilla JavaScript
